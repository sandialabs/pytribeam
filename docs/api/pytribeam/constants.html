Constants Module

This module contains various constants and conversion factors used throughout the software. The constants are organized into two main classes: Constants and Conversions.

Classes

Constants : NamedTuple A NamedTuple containing various constants related to software versions, log files, beam parameters, detector parameters, laser parameters, FIB parameters, stage parameters, mapping parameters, test suite parameters, and error message display parameters.

Conversions : NamedTuple A NamedTuple containing various conversion factors for length, time, voltage, current, and angle.

286class Conversions(NamedTuple):
287    """
288    A NamedTuple containing various conversion constants for length, time, voltage, current, and angle.
289
290    Attributes
291    ----------
292    MM_TO_M : float
293        Conversion factor from millimeters to meters.
294    UM_TO_M : float
295        Conversion factor from micrometers to meters.
296    M_TO_MM : float
297        Conversion factor from meters to millimeters.
298    M_TO_UM : float
299        Conversion factor from meters to micrometers.
300    UM_TO_MM : float
301        Conversion factor from micrometers to millimeters.
302    MM_TO_UM : float
303        Conversion factor from millimeters to micrometers.
304
305    US_TO_S : float
306        Conversion factor from microseconds to seconds.
307    S_TO_US : float
308        Conversion factor from seconds to microseconds.
309    S_TO_NS : float
310        Conversion factor from seconds to nanoseconds.
311    NS_TO_S : float
312        Conversion factor from nanoseconds to seconds.
313    US_TO_NS : float
314        Conversion factor from microseconds to nanoseconds.
315    NS_TO_US : float
316        Conversion factor from nanoseconds to microseconds.
317
318    KV_TO_V : float
319        Conversion factor from kilovolts to volts.
320    V_TO_KV : float
321        Conversion factor from volts to kilovolts.
322
323    UA_TO_A : float
324        Conversion factor from microamperes to amperes.
325    NA_TO_A : float
326        Conversion factor from nanoamperes to amperes.
327    PA_TO_A : float
328        Conversion factor from picoamperes to amperes.
329    A_TO_UA : float
330        Conversion factor from amperes to microamperes.
331    A_TO_NA : float
332        Conversion factor from amperes to nanoamperes.
333    A_TO_PA : float
334        Conversion factor from amperes to picoamperes.
335    PA_TO_NA : float
336        Conversion factor from picoamperes to nanoamperes.
337    PA_TO_UA : float
338        Conversion factor from picoamperes to microamperes.
339    NA_TO_UA : float
340        Conversion factor from nanoamperes to microamperes.
341    NA_TO_PA : float
342        Conversion factor from nanoamperes to picoamperes.
343    UA_TO_NA : float
344        Conversion factor from microamperes to nanoamperes.
345    UA_TO_PA : float
346        Conversion factor from microamperes to picoamperes.
347
348    DEG_TO_RAD : float
349        Conversion factor from degrees to radians.
350    RAD_TO_DEG : float
351        Conversion factor from radians to degrees.
352    """
353
354    # length
355    MM_TO_M = 1.0e-3
356    UM_TO_M = MM_TO_M / 1000.0
357    M_TO_MM = 1.0 / MM_TO_M
358    M_TO_UM = 1.0 / UM_TO_M
359    UM_TO_MM = UM_TO_M * M_TO_MM
360    MM_TO_UM = 1.0 / UM_TO_MM
361
362    # time
363    US_TO_S = 1.0e-6
364    S_TO_US = 1.0 / US_TO_S
365    S_TO_NS = S_TO_US * 1000.0
366    NS_TO_S = 1.0 / S_TO_NS
367    US_TO_NS = US_TO_S * S_TO_NS
368    NS_TO_US = 1.0 / US_TO_NS
369
370    # voltage
371    KV_TO_V = 1.0e3
372    V_TO_KV = 1.0 / KV_TO_V
373
374    # current
375    UA_TO_A = 1.0e-6
376    NA_TO_A = UA_TO_A / 1000.0
377    PA_TO_A = NA_TO_A / 1000.0
378    A_TO_UA = 1.0 / UA_TO_A
379    A_TO_NA = 1.0 / NA_TO_A
380    A_TO_PA = 1.0 / PA_TO_A
381    PA_TO_NA = PA_TO_A * A_TO_NA
382    PA_TO_UA = PA_TO_A * A_TO_UA
383    NA_TO_UA = NA_TO_A * A_TO_UA
384    NA_TO_PA = 1.0 / PA_TO_NA
385    UA_TO_NA = 1.0 / NA_TO_UA
386    UA_TO_PA = 1.0 / PA_TO_UA
387
388    # angle
389    DEG_TO_RAD = math.pi / 180.0  # convert degrees to radians
390    RAD_TO_DEG = 180.0 / math.pi
